文件要求怎么理解？

纽约 DMV 办 REAL ID 或 Enhanced 时，通常会要求证明身份、出生日期、合法身份、Social Security 相关信息，以及纽约州地址证明。相比普通 Standard，REAL ID / Enhanced 对文件核验更严格。

地址证明REAL ID 和 Enhanced 一般需要 2 份纽约州地址证明，文件上要显示当前纽约地址。
姓名一致如果护照、绿卡、出生纸、结婚纸姓名不一致，可能要带改名、结婚或离婚证明。
Standard通常文件压力较小，但证件上会标注 Not for Federal Purposes。
Enhanced功能最多，但不是护照，坐国际飞机仍要护照。

华人常见选择建议

我只是考 Learner Permit，要不要直接办 REAL ID？

如果你以后会坐美国国内飞机，建议一开始就考虑 REAL ID，免得以后再跑一次 DMV。如果你已经有护照，也可以先办 Standard，但坐飞机时要记得带护照。

REAL ID 和 Enhanced 哪个更好？

对大多数纽约华人来说，REAL ID 已经够用。Enhanced 多了陆路/海路从加拿大、墨西哥和部分加勒比地区返回美国的功能，但通常有额外费用。若你不常开车过境，未必需要 Enhanced。

Standard 驾照是不是不能开车？

不是。Standard 驾照仍然可以用于纽约州开车。它主要问题是不能作为联邦 REAL ID 用途，例如不能用它坐美国国内飞机或进入部分联邦设施。

有护照，还需要 REAL ID 吗？

不是必须。护照是 TSA 接受的身份证件，可以用于坐飞机。但很多人不想每次国内旅行都带护照，所以会选择 REAL ID 或 Enhanced。

纽约 DMV 中文说明

如果你在纽约准备办驾照、学车纸、Non-Driver ID、REAL ID 或 Enhanced License，最容易混淆的就是三种证件的实际用途。Standard License 是普通纽约驾照，可以开车，但上面会标注 Not for Federal Purposes；REAL ID 是符合联邦标准的驾照或身份证，适合坐美国国内飞机和进入部分联邦设施；Enhanced License 则在 REAL ID 功能之外，还可以用于从加拿大、墨西哥和部分加勒比地区经陆路或海路返回美国。

对纽约华人来说，如果你只是日常开车、买车、保险或处理普通 DMV 业务，Standard 可能够用；如果你经常坐美国国内航班，REAL ID 更方便；如果你经常开车去加拿大或需要边境陆路通行便利，Enhanced 才更值得考虑。无论选择哪一种，去 DMV 前都建议先用官方 Document Guide 检查文件，避免因为地址证明、SSN、姓名不一致或合法身份文件问题白跑一趟。

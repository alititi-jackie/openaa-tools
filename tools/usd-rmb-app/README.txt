美元人民币汇率记录工具 - 完整包

上传方法：
1. 将本压缩包解压。
2. 把 index.html、manifest.json、sw.js 三个文件一起上传到 GitHub Pages 对应目录。
3. 如果已经有旧版本，直接覆盖同名文件即可。

本版优化：
- Excel 导出分为：全部记录、月度记录、年度记录。
- 金额、汇率、人民币等数字右对齐，手机查看更整齐。
- 汇总顺序统一：美金总金额、平均汇率、人民币总金额、记录行数、有记录月份、每月平均美金、每月平均人民币。
- 月度查看默认显示最新有记录月份，不再显示全部月份按钮。
- 按月查看只显示当月汇总，不显示每月平均美金/人民币。
- 复制汇总内容用于微信粘贴发送。

# 美元人民币记录 PWA

静态网页 App，可上传到 GitHub Pages 使用。

功能：
- 日期、美金、汇率输入
- 自动计算人民币
- 本机 localStorage 保存
- 查看全部、按月、按年汇总
- 导出 Excel（WPS 可打开）
- JSON 备份导出/导入

使用：
1. 把本目录所有文件上传到 GitHub 仓库。
2. GitHub 仓库 Settings → Pages → Deploy from branch → main / root。
3. 用手机打开 GitHub Pages 地址。
4. 浏览器菜单选择“添加到主屏幕”。

注意：数据保存在当前手机浏览器本地。换手机、清缓存、换浏览器前请先导出 JSON 备份。

Section A：Social Security 相关

申请时通常需要提供 SSN 相关证明。原始 Social Security Card 可计 2 分；只有号码但没有卡一般不计分。

Section B：生日 / 公民身份 / 合法身份

这里至少通常要有一类能证明出生日期、身份或合法状态的文件。护照、绿卡、EAD 等常见文件在这里计分。

Section C：纽约地址证明 + 额外积分

地址证明文件必须显示你当前纽约州地址，P.O. Box 不接受。电子账单需要打印出来。同一种来源或同一种文件一般只能算一份。

给纽约华人的实用提醒

很多人去 DMV 白跑一趟，不是因为总分不够，就是地址证明不符合要求。去之前最好把文件按三类分好：第一类是身份证明和生日/合法身份文件，第二类是 Social Security 相关文件，第三类是纽约地址证明。

如果你申请 REAL ID 或 Enhanced，地址证明通常要准备两份，而且必须显示当前纽约州地址。银行账单、水电煤账单、工资单、租约、税表等都可能有用，但同一个来源不要重复计算。

如果姓名和护照、绿卡、出生纸、结婚纸上的名字不一致，可能还需要带结婚证、离婚判决或法院改名文件，把名字变化串起来。
